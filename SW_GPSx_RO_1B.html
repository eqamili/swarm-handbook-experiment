SW_GPSx_RO_1B

GPS RINEX Observation data

Thematic areas: GNSS measurements

Applicable missions: Swarm

Applicable spacecraft: Swarm-A, Swarm-B, Swarm-C

Description

The GPSx_RO_1B Products contain the GPS Observation files generated from GPSR data. The product stores the GPS Observation data file in the RINEX 3.00 format generally used for LEO satellites. The description of the format can be found in rinex300.pdf, Section 2 and Appendix A.
