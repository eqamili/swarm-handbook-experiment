SW_MTI_SHA_2C

Spherical harmonic model of the magnetic signature of oceanic M2 tide

Thematic areas: Ocean Tides

Applicable missions: Swarm

Applicable spacecraft: Swarm-A, Swarm-B, Swarm-C

Description

This product contains the representation of a model of the magnetic field of the Lunar M2 ocean tides using spherical harmonic coefficients. The model is estimated from Swarm and ground observatory data using the Comprehensive Inversion scheme within the Swarm Level 2 Processing system.
