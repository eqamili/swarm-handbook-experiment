SW_GPSx_RN_1B

GPS RINEX Navigation data

Thematic areas: GNSS measurements

Applicable missions: Swarm

Applicable spacecraft: Swarm-A, Swarm-B, Swarm-C

Description

The GPSx_RN_1B Products contain the GPS Navigation files generated from GPSR data. The product stores the GPS Navigation message file in the RINEX 3.00 format generally used for LEO satellites. The description of the format can be found in rinex300.pdf, Section 2 and Appendix A.
